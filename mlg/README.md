# mlog GUI (MLG)

mlog GUI is a specialized Java-based integrated development environment designed specifically for Mindustry Logic. It provides a clean interface for writing and managing processor scripts with advanced editing features.

---

## English Version

### Features
* Code Autocompletion: Includes all standard mlog commands (read, write, draw, print, unitcontrol, etc.).
* Syntax Highlighting: Powered by the RSyntaxTextArea engine for better code readability.
* Localization: Full support for external language files (.lng).
* Dark Theme: Integrated dark mode for comfortable long-term use.

### Usage
* To trigger the autocompletion menu, press L-Control + Space while typing.
* The editor supports code folding and anti-aliasing for a smoother visual experience.

### Installation and Requirements
* Requires Java 8 or higher.
* To run the application, the lang/ directory must be located in the same folder as the .jar file.
* If you are compiling from source, ensure that rsyntaxtextarea and autocomplete libraries are included in your classpath (or use Maven to handle dependencies via pom.xml).

### Important Note on Binaries
The standalone .jar file depends on the lang/ folder to display UI strings correctly. When downloading the executable, ensure the lang/ directory is placed in the same directory as the jar.
